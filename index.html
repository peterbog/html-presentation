  • < og > (mindre end og større end)
  • Navn
  • Attributter
<a href="http://disney.com">Et link</a>

Attributter

href="http://disney.com"

  • Navn
  • = (lig med)
  • Værdi, omkranset af " (gåseøjne)
  • = og værdi kan unlades ved nogen attributter

Slut tag

</a>

  • < og > (mindre end og større end)
  • / (skråstreg)
  • Navn
  • Nogen elementer har ikke en krop og behøver derfor ikke slut tag
    I de situationer kan start tagget slutte med / (skråstreg)
    fx <input type="checkbox" />

Krop

En krop kan bestå af tekst og andre HTML elementer

Et <strong>fedt</strong> link

Specialtegn

Alle < bliver opfattet som starten på et tag.
Specialtegn kan skrives med &xx;
< &lt;
> &gt;
" &quot;
& &amp;

Whitespace

Mellemrum og linjeskift ignoreres næsten.

En række whitespace smeltes sammen til ét mellemrum.

HTML dokument


<html>
	<head>
		<title>Hello title!</title>
		<link rel="stylesheet" href="/styles/style.css" />
		<script src="/scripts/script.js"></script>
	</head>
	<body>
		Hello world!
	</body>
</html>
